# AI Playbook

Bu klasör, projede çalışan ajanların hızlı ve tutarlı karar alması için kısa referans dosyalarını içerir.

- `rules.md` : Projeye özgü zorunlu çalışma kuralları
- `skills.md` : Tekrarlanan görevler için kısa uygulama tarifleri
- `errors.md` : Sık görülen hatalar ve doğrudan çözüm adımları

Not: Bu dosyalar kısa tutulur. Yeni bir tekrar eden pattern çıktığında güncellenir.
